CSS

Introduction about CSS:

  • CSS, Cascading Style Sheets, designs the website layout, styles, colors, and etc.
  • CSS is used with HTML to style the content of the web.
  • CSS saves a lot of time and typing, because it can add a layout to many webpages.
  • There are 3 ways to style a HTML document:
    • Inline styling- add styling next to the HTML elements (not recommended because it looks really messy)
    • External styling- add styling using different file, a .css file
    • Embedding styling- add styling in the same HTML document (in this website, we will use this styling)

Basic layout in CSS:

<head>
<style>
body {
background-color: pink;
}
h1 {
color: darkred;
text-align: center;
}
p {
font-size: 25px;
}
</style>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a big paragraph</p>

</body>

This is a heading

This is a big paragraph


Explaination:

  • Styling needs to be in the <head> element
  • Words between the {} are called Declaration
  • h1, p, and body are called element selector
  • color, text-align, background-color, font-size are called Property
  • pink, darkred, center, 25px are called Value
  • There are many selectors, but the most three common ones are:
    • Element selector
    • Class selector
    • ID selector
