JavaScript

Introduction about JavaScript:

  • JavaScript is the most popular programming language in web development.
  • JavaScript adds more functionality to your website.
  • JavaScript is usually activated with HTML elements (such as clicking button).
  • There are 3 ways to style a HTML document:
    • Inline- add JavaScript next to the HTML elements (not recommended because it looks really messy)
    • External- add JavaScript using different file, a .js file
    • Internal- add JavaScript in the same HTML document (in this website, we will use this way)

JavaScript can do many things related HTML and CSS like:
